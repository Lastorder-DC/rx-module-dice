# rx-module-dice
라이믹스 주사위 모듈

## 소개
ringmaster/dicecalc 라이브러리를 이용해서 라이믹스의 글이나 댓글 본문에서 `{dicestring}` 형태를 자동으로 변환하는 모듈입니다.

## 기능
- 글과 댓글 작성/수정 시 주사위 표현식 자동 변환
- 적용할 모듈/제외할 모듈 설정 가능
- 주사위 표현식뿐만 아니라 일반 수식도 지원

## 사용법

### 기본 주사위 표현식
- `{3d6}` - 6면 주사위 3개 굴리기
- `{2d20+5}` - 20면 주사위 2개 + 5
- `{d20}` - 20면 주사위 1개
- `{4d6h3}` - 6면 주사위 4개 중 높은 3개만

### 수식 계산
- `{(4+4)*5}` - 수식 계산
- `{10+20-5}` - 사칙연산

### 결과 형식
입력: `{3d6}`  
출력: `[3d6 → [3d6:5 + 4 + 1] = 10]`

## 설치
1. 모듈을 라이믹스 modules 디렉토리에 복사
2. 관리자 페이지에서 모듈 설치
3. 주사위 모듈 설정에서 원하는 모듈 선택

## 설정
관리자 페이지 > 주사위 메뉴에서:
- 적용할 모듈 선택 (선택하지 않으면 모든 모듈에 적용)
- 제외할 모듈 선택
